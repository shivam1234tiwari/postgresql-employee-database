# PostgreSQL Employee Database Project

## 📌 Project Overview
This project demonstrates how to create and manage an Employee table using PostgreSQL.

It includes:
- Table creation
- 50 employee records
- Basic SQL queries
- Department-wise salary analysis

## 🛠 Technologies Used
- PostgreSQL
- SQL

## 📂 Project Structure
- schema.sql → Contains table creation script
- insert_data.sql → Contains 50 employee records
- queries.sql → Contains sample queries

## 🚀 How to Run

1. Open pgAdmin or psql terminal
2. Run schema.sql
3. Run insert_data.sql
4. Execute queries.sql

## 📊 Features
- Primary Key (SERIAL)
- Unique Email Constraint
- Salary with Decimal Precision
- Timestamp Auto Generation
